Markdown Notes

Posted on 2017-12-21   |   In Markdown

这是一份个人使用Markdown的笔记总结，文档演示完成的功能如下

  • [x] Markdown嵌入HTML标签
  • [x] Markdown文档编写
  • [x] Markdown生成PDF版面

开始之前简单描述Markdown，Markdown 是一种轻量级的「标记语言」，它的优点很多，目前也被越来越多的写作爱好者，撰稿者广泛使用。其语法十分简单。下面文档会总结到，学习成本也不需要太多，且一旦熟悉这种语法规则，会有一劳永逸的效果。下面列几个优点：

  • 纯文本内容，兼容所有的文本编辑器与字处理软件，可以转化为pdf，word，LaTex等多种格式
  • 可读，直观。适合所有人的写作语言
  • 对于编写嵌入代码的文档，非常友好，易编辑，易修改，易分享

从上面这段html片段中，我想说明的是Markdown语法是兼容HTML语法的，正如其目标一样：成为一种适用于网络的书写语言

###兼容HTML
Markdown语法是无法与html语法相比，准确的描述就是，Markdown语法只是对应html标记的很小一部分，两者的对象领域不同，html划分到发布的一种格式，Markdown则是一种书写的格式，它的对手是纯文本，比较之，可以发现，Markdown能让文档更容易读，容易写，同时又方便我们相互修改，下面介绍我们常用的Markdown语法，涵盖90%的使用场景
